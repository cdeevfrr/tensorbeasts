## Code for TensorBeasts

This is a React-Native app created with Expo.

## Deploys

TensorBeasts is deployed as a web app to github pages, you can use `npm run deploy` to redeploy the public app. 

https://docs.expo.dev/distribution/publishing-websites/#github-pages

See the live app at https://www.todo.com

## Local development

View a local copy of the app without deploying with `npm run web`
